MSDN IDs

This web service provides mappings from the short IDs used on the MSDN website and the canonical names of classes and methods.

To look up an ID simply add it to the end of the services URL. For example, to look up the short ID "zba0x2tx" you would call http://msdnid.alnorth.com/zba0x2tx.

By default the value will be returned as plain text. If the service can't find a canonical name for the ID then it will return an empty response.

Cross-domain

CORS requests are supported using the Access-Control-Allow-Origin: * header. If you need to use it in browsers that don't support CORS then JSONP is also supported, you just need to add a callback parameter on the query string.
